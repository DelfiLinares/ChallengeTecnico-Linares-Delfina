# JuegoContador — Challenge React

Aplicación web desarrollada en React donde el usuario debe hacer la
mayor cantidad de clicks posibles en **5 segundos**, compitiendo contra
su propio récord.

---

## Requisitos previos

- Node.js (versión 18 o superior recomendada)
- npm (incluido con Node)

Verificar instalación:

    node -v
    npm -v

---

## Clonar el repositorio

    git clone URL_DEL_REPOSITORIO
    cd juego-contador

---

## Instalar dependencias

    npm install

---

## Ejecutar el proyecto en local

    npm run dev

Abrir en el navegador: http://localhost:5173/

---

## Uso de la aplicación

1. Presionar **Iniciar juego**
2. Esperar la cuenta regresiva: *Preparados... Listos... ¡Ya!*
3. Durante **5 segundos** hacer la mayor cantidad de clicks posibles en **¡CLICK!**
4. Al finalizar:
   - Se muestra el puntaje de la partida
   - Se actualiza el récord si fue superado
