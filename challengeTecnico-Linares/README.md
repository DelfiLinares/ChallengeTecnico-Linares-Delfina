# Juego de Clicks

Challenge hecho por mi misma sin ayuda de la ia (o con su ayuda en la menor medida posible) basandome en los otros dos jueguitos que hice con ayuda de claude y chatGPT.

---

## como se juega

1. apreta el boton verde que dice **Iniciar**
2. espera la cuenta: Preparados... Listos... Ya!
3. cuando aparece el **Ya!** clickea el boton azul lo mas rapido que puedas
4. cuando se acaba el tiempo te dice cuantos clicks hiciste
5. si superaste tu record anterior te avisa!

---

## como abrir el juego

no necesitas instalar nada, en serio

1. descarga el archivo **juego-clicks.zip**
2. descomprimilo (click derecho > extraer)
3. entra a la carpeta que se crea
4. hace doble click en **index.html**
5. se abre solo en el navegador y listo

> si no se abre solo, arrastras el archivo al navegador y ya

---

## que archivos hay

Es un solo archivo. el html, el css y el javascript estan todos juntos.

---

## tecnologias usadas

- HTML
- CSS
- JavaScript

nada de librerias ni nada raro, puro codigo basico.

---

## cosas que hace el juego

- cuenta regresiva antes de empezar (Preparados / Listos / Ya!)
- timer de 5 segundos
- cuenta los clicks en tiempo real
- guarda el record mas alto de la sesion
- los botones se deshabilitan cuando no se pueden usar
