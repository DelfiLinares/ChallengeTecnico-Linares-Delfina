# JuegoContador -- Challenge React

Aplicación web desarrollada en React donde el usuario debe hacer la
mayor cantidad de clicks posibles en **5 segundos**.

------------------------------------------------------------------------

## Requisitos previos

-   Node.js (versión 18 o superior recomendada)
-   npm (incluido con Node)

Verificar instalación:

    node -v
    npm -v

------------------------------------------------------------------------

## Clonar el repositorio

    git clone URL_DEL_REPOSITORIO
    cd juego-contador

------------------------------------------------------------------------

## Instalar dependencias

    npm install

------------------------------------------------------------------------

## Ejecutar el proyecto en local

    npm run dev

Abrir en el navegador: http://localhost:5173/

------------------------------------------------------------------------

## Uso de la aplicación

1.  Presionar **Iniciar juego**
2.  Durante **5 segundos** hacer la mayor cantidad de clicks posibles en
    **CLICK!!**
3.  Al finalizar:
    -   Se muestra el puntaje
    -   Se actualiza el récord si se supera
